# Blog Test Plan

## Application Overview

This test plan covers the functionality of checking the latest blog article on the LambdaTest e-commerce playground site. It ensures that users can access the blog section and view the most recent article.

## Test Scenarios

### 1. Blog Tests

**Seed:** `seed.spec.ts`

#### 1.1. Check Latest Blog Article

**File:** `check-latest-blog-article.spec.ts`

**Steps:**
  1. Navigate to the blog page
  2. Verify the latest blog article is displayed

**Expected Results:**
  - The blog page loads successfully
  - The latest blog article title is visible
